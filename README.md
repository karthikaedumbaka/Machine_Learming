**Learning NumPy**
==================

Welcome to the **Learning NumPy** repository! 🚀 This repository showcases my journey through mastering **NumPy**, a powerful library for numerical computing in Python. It contains detailed Jupyter Notebooks covering foundational concepts, advanced techniques, and useful tips for efficient data manipulation.

**Repository Structure**
------------------------

📂 **Files and Topics Covered**

1.  **numpy\_1.ipynb**
    
    *   Introduction to NumPy arrays.
        
    *   Array creation (np.array, np.zeros, np.ones, etc.).
        
    *   Indexing, slicing, and reshaping arrays.
        
    *   Broadcasting basics and universal functions.
        
2.  **numpy\_2.ipynb**
    
    *   Advanced operations with np.arange(), np.linspace(), and np.random.
        
    *   Aggregation methods: sum, mean, std, and more.
        
    *   Reshaping arrays and exploring dimensions.
        
3.  **numpy\_tricks.ipynb**
    
    *   Boolean indexing and conditional operations.
        
    *   Sorting and searching with np.argsort() and np.where().
        
    *   Optimization tips for memory and performance.
